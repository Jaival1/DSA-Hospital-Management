# DSA-Hospital-Management
The Following Code(Hospital Management System) Enables Users as a patient and employees/staff to log in and sign giving many menu options. The options for patients include login signup, checkout, information etc. The menu options for staff include signup, login, and many other options. This Project is a miniature version of the actual Hospital Management System.
